The **QuickCart Dashboard 🛒** is a front-end e-commerce web application designed to demonstrate how real-world online shopping platforms work using core web technologies like HTML 🌐, CSS 🎨 (Bootstrap), and JavaScript ⚡. The application fetches product data from an external API 🔗 and dynamically displays it in a responsive grid layout 📱💻 using Bootstrap cards. Each product includes details such as title 🏷️, price 💰, image 🖼️, and category 📂. Users can interact with the dashboard by searching products in real-time 🔍, filtering them based on categories 🗂️, and sorting them by price in ascending ⬆️ or descending ⬇️ order. Additionally, clicking on a product opens a modal view 🪟 that shows detailed information like description 📄 and rating ⭐.

The application also includes a cart system 🛍️ where users can add products ➕, and the cart count updates instantly 🔢 in the navigation bar. The selected items are stored in the browser’s localStorage 💾, ensuring that the cart data persists even after page refresh 🔄. A separate cart page 📃 displays all added products, allows users to remove items ❌, and calculates the total price 🧾. The project follows a modular and reusable approach ♻️ by separating logic into different JavaScript functions, making it clean and scalable 📈. It also includes error handling ⚠️ using try-catch blocks and shows a loading indicator ⏳ while fetching data from the API. Overall, this project helps in understanding API integration 🔌, DOM manipulation 🧠, event handling 🎯, and responsive UI design 📐, while also preparing a strong foundation for transitioning into React-based development ⚛️.
